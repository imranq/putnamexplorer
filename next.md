# Putnam Site Next Features

## Completed
- Full keyboard navigation:
  - `j`: next problem
  - `k`: previous problem
  - `/`: focus search
  - `s`: toggle solution collapse

## High-Value Next
- Topic drill mode:
  Generate random problems from selected topic + difficulty.

- Try-first timer:
  Start timed attempt (20/40/60 min) before hints/solution reveal.

- Progressive hint reveal:
  Show hint 1, then hint 2, then hint 3 on demand.

- Favorites:
  Star problems and filter by favorites.

- Solved tracking:
  Mark solved/unsolved/in-progress and filter by status.

- Practice set builder:
  One-click sets (e.g., 5 hard geometry, random A-session set).

- Similar problems:
  Show related problems by embedding/metadata similarity.

- Export tools:
  Export current problem/set to Markdown, PDF, and Anki-ready format.

- Advanced search syntax:
  Support query filters like `year:2018 topic:combinatorics has:solution`.

- Difficulty calibration:
  Add user votes to calibrate difficulty labels over time.

- Mobile study mode:
  Single-column focus mode with larger typography and sticky controls.
